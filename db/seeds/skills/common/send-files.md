# Sending Files

When you create files (screenshots, documents, reports):
1. Save them to the workspace/ directory
2. Tell the user the filename and what it contains
3. The user can access files through the dashboard

## File naming
- Use descriptive names: `competitor_analysis_2026-04-08.png`
- No spaces in filenames, use underscores
- Include date for reports

## Screenshots
- Save to workspace/screenshots/
- Include a description of what the screenshot shows

## Documents
- Save to workspace/documents/
- Markdown format preferred for text documents
- CSV for tabular data
